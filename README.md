# Sales Performance Dashboard

This project is an Excel-based interactive dashboard built using Pivot Tables, Charts, and Slicers.

## 📊 Project Overview
The dashboard provides insights into:
- Sales by Category
- Profit by Region
- Monthly Sales Trends

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## 🔍 Key Insights
- Technology category has the highest sales
- Office Supplies has the lowest sales
- Sales show an increasing trend over time
